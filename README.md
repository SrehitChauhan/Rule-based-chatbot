# Rule-Based Chatbot 🤖

This is my first Python project.

I created this chatbot to practice Python basics like dictionaries, functions, loops, conditions, and file handling.

## Features

1.- Answers predefined questions.
2.- User can add new questions and answers using the modify command.
3.- Saves new data in op.txt so it is available after restarting the program.
4.- Uses a dictionary to store chatbot memory.
5.- Greets the user based on the current time.

## Technologies Used

- Python
- Dictionaries
- Functions
- Loops
- File Handling

## How to Run

1. Download or clone this repository.
2. Make sure chatbot.py and op.txt are in the same folder.
3. Open the folder in VS Code.
4. Run:

bash
python chatbot.py


## Commands

- Type your question to get a reply.
- Type modify to add a new question and answer.
- Type bye to exit the chatbot.
